# Advanced CSS

#### Basic css reset

```
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}
```

#### Setup Project wide font definitions

```
body {
    font-family: "Lato", sans-serif;
    font-weight: 400;
    font-size: 16px;
    line-height: 1.7;
    color: #777;
}
```
